# YouTube Ad Revenue Analytics

A full data analysis project exploring YouTube ad revenue trends using SQL, Excel, and Looker Studio.
Built entirely on iPad Air using cloud-based tools.

## Status
🚧 In progress

## Tools
- Excel (iPad) — data cleaning and feature engineering
- PostgreSQL via Supabase — SQL analysis
- Looker Studio — dashboards and visualization
- GitHub — version control and portfolio hosting

## Dataset
YouTube Revenue Data 2018–2021 via Kaggle
